Telnet Daemon

The Telnet Daemon bundle provides a simple telnet server that allows users to telnet into the Oscar shell, allowing you to shut down the Shell TUI and run with all output redirected.

Quick Start

How to use the Telnet Daemon:

Simply install and start the bundle. You will then be able to telnet into Oscar. 
$ telnet manjar 6666
...
Connected to localhost.
Escape character is '^]'.

Welcome to Oscar
================

Remote connection from localhost
Local time is Thu Sep 29 14:23:26 CLT 2005
Be careful. Hit ^D to exit.

Bundle Properties

edu.gemini.util.telnetd.port
This property may be set to specify the telnet port. Otherwise it will default to 6666.
